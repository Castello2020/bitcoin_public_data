SparkleShare – Un clone de Dropbox open source qui fonctionne !

Bon on commence tôt ce matin, car j'ai enfin trouvé un remplaçant open source qui fonctionne et qui roxx à Dropbox ! Merci MrBoo ! L'outil s'appelle SparkleShare et va vous permettre enfin de synchroniser vos documents entre plusieurs ordinateurs, en utilisant votre propre serveur.


Pour fonctionne SparkleShare a besoin d'un serveur Git. Vous pouvez configurer le votre ou alors utiliser celui de Gnome, ou Github...etc. Au niveau des clients, SparkleShare existe pour Linux et Mac (et très bientôt Windows).

Comme j'ai un serveur chez OVH qui ne sert pas à grand chose, avec un bon paquet de place dessus, j'ai fait un petit test. Tout d'abord, il vous suffit d'installer Git sur le serveur :

sudo apt-get install git-core openssh-server

Ensuite, il suffit d'initialiser un dépôt :

git init --bare MON_DEPOT.git

Après, c'est au niveau de SSH que ça se passe. Vous devez autoriser votre ordinateur à accéder au serveur, avec une clé SSH. Les tutos sont ici.

Une fois la clé SSH en place, il suffit de lancer SparkleShare et de saisir les infos relative au serveur ou à votre compte Github, Gnome...etc

SparkleShare se loge ensuite dans votre barre des tâches et vous propose d'accéder à un répertoire sur votre ordinateur où chaque fichier que vous placerez dedans sera automatiquement synchronisé avec le serveur Git.

A vous ensuite de réaliser la même config sur d'autres ordinateurs pour partager les mêmes fichiers entre plusieurs machines. Bon, ça fait un moment que Git existe et qu'il y a tout un tas de clients logiciels ou web pour faire ce genre de choses, mais c'est la premier fois qu'un outil (open source) est "packagé" pour faire de la concurrence à Dropbox. En tout cas, pour ceux qui auraient pris peur avec les dernières TOS ou qui veulent tout simplement conserver la main sur leurs données, SparkleShare est un cadeau du ciel !
