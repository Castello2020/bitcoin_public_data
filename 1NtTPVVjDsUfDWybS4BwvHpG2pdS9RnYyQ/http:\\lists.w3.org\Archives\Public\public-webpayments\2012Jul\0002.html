Okay, the first 4 videos are posted!

http://open-transactions-tv.github.com/

These videos cover:

1. automake, the data folders, intro to the new "opentxs" command-line
interface, and the client-side scripting.
2. opentxs CLI: adding server contracts to the wallet, issuing currencies.
3. opentxs CLI: account-to-account transfer, and (server-side)
generating a new cash mint.
4. opentxs CLI: withdrawing / depositing cash


---------------------------------

5. opentxs CLI: Write / deposit cheque.
6. opentxs CLI: Misc tricks.
7. "Moneychanger" GUI: Basket currencies: how to create, + how to
exchange in/out.

---------------------------------

Also coming soon (already recorded):

8-10: The "two-way trade" smart contract is demoed, with receipts going

11-13: The "escrow" smart contract, in depth.
