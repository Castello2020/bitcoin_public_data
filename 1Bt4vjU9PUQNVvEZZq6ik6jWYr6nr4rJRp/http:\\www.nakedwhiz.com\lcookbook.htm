First of all you are going to need the Living Cookbook program. Click on the link to visit their website. There you will be able to download a free trial version of the program. It will allow you to use the program 30 times before you must register to continue using the program. NOTE: Use coupon code "BIGGREENEG" (note only one "G" on the end!) to obtain a $3.50 discount off the normal price when ordering! The second thing you will need is a program that can deal with .ZIP files. Winzip is one that you use. You can download a free trial evaluation copy that will work for 21 days before you have to register.

If you look around, you should also be able to find some freeware programs that will handle .ZIP files.

The final things you need are the following files which you can then import into The Living Cookbook. You can find information how to do this by reading the help information that is in The Living Cookbook.
