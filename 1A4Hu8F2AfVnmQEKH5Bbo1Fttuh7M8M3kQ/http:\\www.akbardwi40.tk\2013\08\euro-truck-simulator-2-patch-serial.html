Minimum System Requirements

Windows XP/Vista/7, Dual core CPU 2.4 GHz
2 GB RAM, graphics card with 256 MB memory
(GeForce 7600 GT-class equivalent or better)

Recommended System Requirements

Windows 7, Dual core CPU 3.0 GHz
4 GB RAM, graphics card with 1024 MB memory
(GeForce GTS 450-class equivalent or better)

Screenshoot : 
 

Download : Euro Truck Simulator 2 

Cara Install : 
  •  Jalankan file setup yg sudah di download tsb
  •  Jangan rubah directory default penginstalannya
  •  Instal gamenya sampe selesai
  •  Karena master download tersebut sudah include patch 1.2.5.1, maka setelah penginstalan selesai langsung jalankan game
  •  Ketika baru masuk game akan di minta Serial Number, Pakai ini:
 V32UA-3HL03-5AEE3-VC3KK-460JM
JX3KK-6YYP2-M6DA1-A1V3T-XDTPD
SQV58-CEN4U-UYZNS-9W0CP-AYRZS
NX2FL-1KQXP-L07R5-T0MMP-YELK9
95TCP-7KXFL-K3PDR-CJF0U-CC17V


  • Pilih salah satu dari Serial Number tersebut, copy dan pastekan secara teliti, jangan sampe ada yg salah dalam memasukan SN tsb.
  • Setelah memasukan SN, klik "Activate Game", setelah itu akan ada bacaan "thank for purchase this game copy bla..bla..bla.. " itu artinya key valid & sukses
  • Setelah sukses game siap dimainkan
Dan ini Patch nya bagi yang tidak memakai Master Game diatas atau game nya belum di patch, bagi yg sudah memakai master game diatas, tidak usah mendownload lagi patch ini
Patch 1.2.5.1
  •  Jalankan patch sampe selesai
  •  Bagi PC/Laptop yg spek nya pas-pasan, selama patch berjalan jangan lakukan kegiatan apapun dgn PC/Laptop (di diemin/tinggal aja sampe patch selesai)
  •  Setelah Patch sukses, buka folder C:\Program Files\Euro Truck Simulator 2\bin\win_x86 , klik kanan pada file eurotrucks2.exe , lalu klik "Properties -> Details"
  •  Lihat di baris "Product Version", apakah tertulis 1.2.5 ?
  •  Kalau sudah tertulis 1.2.5 berarti game sukses di patch, kalau belum, ulangi proses patch

Cara Alternativ Update ke 1.2.5.1 (bagi yang suka diminta serial atau error") 

Berikut langkah" update ke 1.2.5.1
1. Download Patchnya di sini (bagi yg sudah download master game include patch 1.2.5.1 , langsung loncat ke step 3)
2. Install Patch yang barusan didownload
3. Jangan masuk ke game dulu !!, Block dulu koneksi ETS 2 lewat firewall

(Cara Blok Koneksi ) :
  •  Buka Windows Firewall di control panel
  •  Klik Advance Settings
  •  Klik Inbound Rules, trus klik New Rule
  •  Klik Program, trus klik Next
  •  Klik Browse, arahkan ke lokasi eurotrucks2.exe (contohnya = C:\Program Files (x86)\Euro Truck Simulator 2\bin\win_x86), lalu klik Next
  •  Klik Block Connection, klik Next, klik Next lagi.
  •  Kasih Nama & Description ngasal aja, trus klik finish
  •  Lakukan hal yg sama pada Outbound Rules 
4. Masuk ke Game , Enjoy !
