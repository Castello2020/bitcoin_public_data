Dragora GNU/Linux-Libre

Introduction

Dragora is a trustworthy GNU/Linux-Libre distribution based on the concept of simplicity with the goal of being a multi-purpose operating system. Dragora respects the freedom of the user with the values of free software and provides control to those who use it. It is developed entirely by volunteers and it is published under the terms of the GNU General Public License.

Communication Channels (How to Get Help)

Forum

Our community forum at LinuxQuestions.org. A good place where the users of Dragora come to speak or ask for help.

Mailing List

To discuss or inform about bugs in Dragora, the mailing list dragora-bug@nongnu.org is the best place. Please, send your messages in plain text, including a descriptive line in the subject of the email. If all the subjects are "error in Dragora" it's impossible to tell them apart.

The mailing list archive is available at http://lists.nongnu.org/mailman/listinfo/dragora-bug

IRC

For a real time chat, you can join #dragora with an IRC client on the server irc.freenode.net. Also via web by visiting http://webchat.freenode.net/?channels=dragora

Social Network

The group, on the free social network Identica.

How To Help

Some useful advice on how to contribute:

Usage

The more people use and try versions of Dragora, more people can find and report bugs. This helps to increase the stability of each new version.

Development

You can help detecting a malfunction on a tool of the project or related to the project; you can browse the source code of an application and check if something can be improved. If you do not understand anything about programming, you can contribute by detecting spelling mistakes in miscellaneous messages; you can help with ideas or by introducing something that is really necessary; you can help detect an application or documentation that is not free,or a free application that encourages the usage of nonfree tools.

Documentation

Documentation is the help that every free software project needs. A well documented system allows developers and users to understand and communicate things better.

If you want to write any kind of documentation, make a suggestion or have constructive criticism, please, use the communication channels.

Translation (website)

To work with the sources of the web site, get the sources.

Donate

Dragora GNU/Linux-Libre is a redistributable free operating system made by volunteers:

A project like Dragora demands a great deal of time, effort and sacrifice. Some of the tasks are: coding, documenting, translating, maintaining, spreading the word. Also testing and helping users (answering support questions). Other important things are: fulfilling the basic needs (since this is a full time job for the maintainer, in charge of the project), paying web hosting, electricity, internet connection, paying direct contributors, etc.

For more information, read the section on Donations.
