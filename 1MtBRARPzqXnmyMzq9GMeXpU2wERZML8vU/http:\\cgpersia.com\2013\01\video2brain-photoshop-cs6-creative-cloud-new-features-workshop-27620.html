Learn What’s New and How It Affects You
If you’re an Adobe Creative Cloud user, you’ll find that the Photoshop 13.1 update gives you access to some exclusive new features. In this workshop Chad Chelius and Ellery Connell take you on a tour of these enhancements, which focus on file handling and web features. You’ll learn how to edit nondestructively with the Blur Gallery and Liquify filter using new Smart Object support, create conditional actions, copy CSS data to a web page, and more. Also included are tutorials on the new 3D options in Photoshop Extended.
In this workshop you’ll be introduced to the new and enhanced features in the Photoshop 13.1 Adobe Creative Cloud update.
