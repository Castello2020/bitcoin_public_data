Thinkbox Frost 1.0.5.44250 for 3DS Max

FROST is a compound geometry object for 3ds Max that can be used to generate a single mesh from particles, vertex clouds, object positions, point data files or a combination thereof using various iso-surface or geometry cloning techniques.

Install (read carefully!)

1. Install Frost as administrator

2. copy the cracked dlo to the correct folder
ie C:\Program Files\Thinkbox\Frost\3dsMax2010\x64
_________________________________________________________________________________________

 Please Note:

the max 9 dlo will work with max 2008
Also the max 2010 dlo will work with max 2011

If you have multiple versions of max use the “add frost to max utility” located in the bin folder
or by clicking start>all programs>thinkbox>Frost 1.0> add frost to max

__________________________________________________________________________________________

3. Copy Frost-CreateFrost.mcr to C:\Program Files\Thinkbox\Frost\Scripts

4. copy FrostIni.ms to your start up scripts eg. C:\Program Files\Autodesk\3ds Max 2011\Scripts\Startup

5. startup max as administrator

6. Setting up your toolbar (optional but recommended)

O. Go to Main Menu > Customize > Customize User Interface

O. Select the desired UI area (Keyboard, Toolbar, Menu, QuadMenu tab) to customize

O. Locate the “Frost” category in the Category drop-down list

O. Drag the Frost icons to the respective Toolbar, Menu or QuadMenu or assign to Keyboard Shortcuts.
Optionally, locate the MacroScript “Sphere Gizmo (Atmospheres)” in the “Objects Helpers”
category and drag it into the same toolbar or menu or assign a keyboard shortcut.
This will come handy when using the AutoFrost feature described further on this page.

O. Save the Customized User Interface

7. Flux’d!
