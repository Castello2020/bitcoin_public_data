Gnomon Master Class 2011 – Stylized Visual Effects Animation ( Project Files Added ) This class will demonstrate how to use principals of animation to create a more compelling and stylized visual effects sequence. Using a plethora of 3D applications, including 3ds Max, Maya, Houdini, Vue and FumeFX, Slim will emphasize and build upon many animation tricks and techniques, focusing on shapes, color and motion semantics. The main project will be a large scale “Magical Effect” shot. Through execution of the main project, supplemental examples will illustrate the principals used including stylized fluid simulations and ultra-stylized structure destruction.
