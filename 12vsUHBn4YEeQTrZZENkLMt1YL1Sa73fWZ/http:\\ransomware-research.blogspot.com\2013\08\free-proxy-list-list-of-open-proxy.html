Proxy Anonymity Levels


  • Transparent - Web server knows your IP address and it knows that you're using a proxy
  • Anonymous - Web servers does not know your IP address, but it knows that you're using a proxy;
  • Elite - Web server does not know your IP address, nor is it aware that you're using proxy.

How to use Proxy Servers?

Your browser can be configured to use a proxy, and proxy use is supported by all the major web browsers such as Internet Explorer, Mozilla Firefox, and Google Chrome.
