Re: When to use BaseGameActivity and when not to?

by alex » Thu Sep 23, 2010 11:04 pm

I haven't done it, but I would suggest you keep your login activities - which I presume you've used the xml layout for.
Your main game activity should use AndEngine and implement a BaseGameActivity

So your login screen would intend to launch the main game activity (that implements BaseGameActivity).

AndEngine does not have client/server stuff built in - just use the Android http communication functions
