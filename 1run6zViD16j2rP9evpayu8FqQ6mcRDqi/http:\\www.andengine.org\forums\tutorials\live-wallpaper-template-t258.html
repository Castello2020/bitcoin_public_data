Well, in theory you just have to swap the Camera's Width and Height Values around (as thats all you have done when you rotated...).

Check the developer.android.com website for more information on screen sizes etc. I hope to do a tutorial on it at some point when I have more experience in handling them. As of yet though, there is not an Example for AndEngine on changing the orientation of the game though (mainly cos you may not do this in most games... maybe in puzzle games you would though). Im sure someone else can beat me to posting something up if they are kind enough though.

Let me know how you go with it though.

Note: I have a Desire, so its got Sense on it, and therefore I cannot change the orientation of the Live Wallpaper!
