Getting started with AndEngine ...

by Nicolas Gramlich » Tue Jun 15, 2010 1:31 pm

Hello Community,

as you have opened this topic, you are probably interested in getting started with AndEngine - good choice !

You might wonder where to download a ready-to-go sample project
What you are looking for are probably the official AndEngine-Examples. The place to go for the most recent Sourcecode for that project is here:
http://code.google.com/p/andengineexamples/source/checkout

While I strongly encourage you to check out that project (so you can easily pull changes from the repository of the Examples and see if there is something new or what has changed), you don't need to do so. (Mercurial Eclipse Plugin)

At lease you should browse the Sourcecode and get a quick overview on the examples - you can do that here:
http://code.google.com/p/andengineexamples/source/browse/#hg/src/org/anddev/andengine/examples


In the case you do not want to check out the Examples or even build AndEngine yourself, but you want to start a new project with AndEngine from scratch, I created the following Screencast to show how this is done within 5 minutes:


For your convenience, the andengine.jar is located here:
http://andengineexamples.googlecode.com/hg/lib/andengine.jar


Let me know if this was helpful

Best Regards,
