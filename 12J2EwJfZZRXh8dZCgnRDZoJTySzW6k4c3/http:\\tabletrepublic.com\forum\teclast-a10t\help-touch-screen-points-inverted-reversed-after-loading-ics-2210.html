    Help! - Touch Screen points are inverted/reversed after loading ICS

    Hello All,

    I recently purchased the Teclast A10t tablet and it was preloaded with Android 2.3. I now have tried 3 versions of Android 4.0.3 ICS loaded and have the same result, the touch screen seems to be inverted/reversed.

    To confirm this I turned on the visual touch screen setting in the developer section of ICS.
    If I touch the Top Left corner I notice the touch point as the Bottom Left Corner. This is the same behavior for center and right touch positions. Also I noticed if I swipe from left to right it rotates to the left instead of the right. This is the same for up & down swipes.

    I loaded the following versions via links from this forum:
    A10t(5DM3)_android4.0.3V1.02
    A10t(5DM3)_android4.0.3_DCD7
    A10t(5DM3)-android4.0.3-V1.11.rar

    The tablets touch functions were working correctly on 2.3 but on 4.0 it does not.
    Also wanted to mention that when i connect a usb mouse, the cursor performs correctly.

    Is there a way to calibrate the touch screen points?
    Is there a step I missed? (I used livesuit v1.09)

    for reference I found this video on youtube on a different brand tablet but same behavior on ICS.
    D70GT inverted touch problem - YouTube

    *UPDATED - here is my link with images and videos of the problem.
    http://www.flickr.com/photos/46132020@N08/


    Any suggestions would be helpful.
    Thanks & Regards
