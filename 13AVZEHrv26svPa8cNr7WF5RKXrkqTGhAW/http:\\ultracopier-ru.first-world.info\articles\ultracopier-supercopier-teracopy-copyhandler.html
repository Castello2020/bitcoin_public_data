Сравнение ultracopier, supercopier, teracopy и copy handler


Я хотел, чтобы мой продукт завершить сравнить все копировальные аппараты из существующих файлов. Возьмите его на более личной точки зрения, поэтому я по сравнению ultracopier, supercopier, teracopy и copyhandler

Все здесь:
  • Meilleur que Teracopy
  • Лучше, чем SuperCopier
  • Лучше, чем CopyHandler
  • Это быстро
  • Это красиво
Я сравниваю фактические функции и ценности не являются субъективными.

Ultracopier


Преимущества ultracopier по отношению к другим копировальных аппаратов, на мой взгляд:
  • Multi-plateforme (linux, mac et windows)
  • Написанный в C + +, так что каждый может развиваться
  • Доступные в различных исполнениях (нормальный, 64Bits, portable, debug)
  • Написанная в Qt для всех выгоды от этой библиотеки (перевода, темы, CSS, HTML в интерфейс плагина, ...)
  • Попытки стандартизировать вещи и бассейн на код, и максимум Библиотека внутри, так и на других копирах
  • Повышенная безопасность передачи данных
  • Очень хороший план и технический документ или упрощенную, хорошие сообщества и очень отзывчивы разработчиков
  • Хороший обычай сделал сайт и платформу для создания большого сообщества
  • Регулярно обновляется, git версия доступна для последней версии еще не опубликованы
Недостатки:
  • Ошибки на сайте, не очень английский английский
  • Очень немногие пожертвований, поэтому не могу посвятить себя полностью ultracopier
  • Некоторые переводы осуществляются с помощью переводчика Google

Supercopier


Преимущества SuperCopier по отношению к другим копировальных аппаратов, на мой взгляд:
  • Многолетний
  • Прогулка под вино, если вы бросаете копии вручную
  • Open source
Недостатки:
  • Написанная в Паскале, несколько разработчиков могут помочь

Teracopy


Преимущества TeraCopy по отношению к другим копиров, на мой взгляд:
  • Параметры адаптации копии в зависимости от источника средств массовой информации и назначения
Недостатки:
  • плату
  • Небольшое изменение
  • Нет опций для копирования окна
  • Не работает под Windows 8 (ненадежности и плохой профилактики ошибке/аварии)
  • Это окна
  • Утилита для управления буфером размер ползунка отображается на интерфейсе?

Copy handler


Преимущества копию обработчик по отношению к другим копировальных аппаратов, на мой взгляд:
  • Написанный в C + +, так что каждый может развиваться, но некоторые конкретные студии окон и визуальных обязательным
  • Является эффективным
  • Маленькое окно возле лотка, который может быть более подходящим для некоторых
Недостатки:
  • Только под окнами и были проблемы с новыми окнами, но это исправляется
  • Старые версии достаточно (было сделано главным образом в середине 2009 года), только сохранила, не планируется новая версия
  • Интерфейс достаточно печь и всю полезную информацию для разработчиков, которые хотят оптимизировать
  • Существует не портативная версия
