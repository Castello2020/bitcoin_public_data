Perbandingan Ultracopier, supercopier, teracopy et copy handler


Saya ingin membandingkan produk saya untuk penuh semua file yang ada mesin fotokopi. Aku mengambilnya untuk lebih pribadi, baik dalam perbandingan Ultracopier, supercopier, teracopy et copyhandler


Ultracopier


Ultracopier manfaat dibandingkan dengan aplikasi lain adalah untuk saya:
  • multiplatform(Linux, Windows y Mac)
  • Ditulis dalam C + + untuk seluruh dunia untuk mengembangkan
  • Tersedia dalam banyak versi (normal, 64Bits,portabel, debug)
  • Ditulis dalam Qt untuk mendapatkan semua manfaat dari lib ini (terjemahan isu, CSS,HTML Antarmuka, plug-in, ...)
  • eh mencoba untuk membakukan hal dan berbagi dengan kode, liberalisasi internal yang maksimum dan mesin fotokopi lainnya
  • Peningkatan kewaspadaan untuk keselamatan transfer data
  • Sangat baik peta jalan dan makalah teknis atau disederhanakan, masyarakat yang baik pengembang dan sangat sensitif (Saya memiliki alat untuk pemberitahuan instan kinerja tinggi)
  • baik dan situs sebagai platform untuk membangun sebuah komunitas besar
  • Diperbarui secara teratur, tidak svn versi yang tersedia untuk versi terbaru masih buron
kekurangan:
  • Kegagalan di situs(tidak hanya baik di mana-mana),sangat tidak Inggris
  • Sumbangan sangat sedikit, jadi saya tidak dapat sepenuhnya mencurahkan Ultracopie
  • Beberapa terjemahan yang dibuat melalui Google Translator,tetapi memfasilitasi sistem terjemahan akan harus berubah

Supercopier


Tetangga supercopier manfaat mesin fotokopi lain dalam pendapat saya:
  • Panjang
  • Ini berjalan dengan anggur, jika Anda melempar salinan tangan
  • Open source
kekurangan:
  • Ditulis dalam Pascal, pengembang apapun dapat membantu

Teracopy


TeraCopy manfaat tetangga lain mesin fotokopi menurut pendapat saya:
  • Salin parameter pas menurut sumber media dan target
kekurangan:
  • No gratis
  • Perubahan sedikit lama
  • Tidak funcionna dengan jendela 8 (crash)
  • Tidak ada pilihan untuk menyalin jendela
  • Jendela
  • Alat untuk mengelola ukuran buffer untuk slider disorot dalam interface?

Copy handler


Manfaat lain mesin fotokopi salinan tetangga penangan menurut pendapat saya:
  • Ditulis dalam C + + untuk seluruh dunia untuk mengembangkan, tetapi beberapa khusus untuk Windows dan Visual Studio membutuhkan
  • hal ini efisien
  • Jendela kecil untuk system tray siap beberapa lebih cocok untuk beberapa
kekurangan:
  • Windows saja dan punya masalah dengan jendela baru, tetapi tetap
  • Versi mendapatkan tidur yang cukup (itu sebagian besar pertengahan-2009), hanya memegang, tidak ada versi baru yang direncanakan
  • Antarmuka informasi oven panas cukup dan berguna untuk pengembang yang ingin mengoptimalkan
  • Ada versi portabel
