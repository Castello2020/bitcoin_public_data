Supercopier 3 et Ultracopier
Bonjour,
J'ai besoin de beta testeur pour Supercopier 3 en 2013. Pour plus de détails voir mon post de supercopier sur mon blog.
J'ai fait quelque révision mineur d'ultracopier pour corrigé les erreurs de jeunesse (à fois d'ultracopier et de Qt). J'attends la révision de Qt, pour corrigé des truc mineur. Ensuite je m'attaquerai à la nouvelle version, aux nouveaux plugins.
Merci de soutenir ces projets (financer le développement), cela me permet de faire ces projets en tant que professionel et non comme amateur.
EDIT: version de Supercopier 3.0.0.0 publié
Changements:
  • gestion native et complète de l'unicode
  • prise en charge de l'UAC pour windows
  • améliorations diverses dans le code
  • amélioration du chargement des DLL d'interception de la copie
  • amélioration du packaging
  • réécriture du code pour lazarus
  • amélioration des performances
EDIT2: Si vous avec d'autre traduction pour Supercopier que: Allemand, Espagnol, Francais, Portugé, Tieng Viet. Envoyé les moi, merci.
