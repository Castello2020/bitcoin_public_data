Page des dons


Un paiement ou un don, vous pouvez demander que l'argent soit investi sur les points qui vous tiennent à coeur.
Dons sont pour:
  • Open source:
    • Intégration dans l'explorateur de votre choix
    • Le développement d'ultracopier
    • Performance et code plus propre
  • Propriétaire:
    • Passer le code propriétaire en open source (GPL3)


Achat sont pour:
  • Open source:
    • Continuer à améliorer la base
  • Propriétaire:
    • Développement spécifique à votre utilisation
    • Améliorer les plugins déjà existant
