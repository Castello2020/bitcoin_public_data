  RouteSplit




• RouteSplit is a AppleScript Studio application for Mac OS X. With RouteSplit you can add a static route, test it and optionally install a launchd startup script in your system in order to preserve the route settings after reboots.

• RouteSplit e' un'applicazione che consente di aggiungere una route statica, di testarla e di configurare launchd per mantenere le impostazioni anche dopo un reboot.
