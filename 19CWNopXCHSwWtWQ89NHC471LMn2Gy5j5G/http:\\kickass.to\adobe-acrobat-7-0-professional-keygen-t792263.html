It's working fine, after following Valor32 hints.

1. Install Adobe Acrobat
2. Enter serial number you generated using keygen folder
3. Run Adobe and Click Activate, and then when it says you cannot activate it online, chose Phone Activation
4. There you'll see Activation Number, and empty Authorization Code fields
5. Copy that Activation number into Keygen, click "generate authorization code", and then copy that authorization code from keygen back to Adobe in filed under (4)
