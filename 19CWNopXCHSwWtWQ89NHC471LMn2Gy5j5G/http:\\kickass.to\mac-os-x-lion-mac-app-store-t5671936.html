It's genuine Lion. To install, do the following:
1. Right Click on the icon "Install Mac OS X Lion"
2. Select "Show package contents"

5. Click on "Install Mac OS X Lion"
6. Done
During the install process it will download additional files if you are connected to the Internet (just as someone already mentioned).
The install process took about 40 minutes on an iMac 2.8 GHz i7. Everything works great, all will upgrade your stuff to the new Lion apps.
