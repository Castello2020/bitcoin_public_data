ABOUT SVF and MD5:

Dear Torrent Users, pay attention that big torrent files can be downloaded with errors - e.g. you may not have the complete torrent file, no matter what your torrent client program claims. Uploaders often put to the torrent a specific check-file type .SVF or .MD5. This is data verifying file, which contains data about one or more files in the torrent. Once run with the appropriate program, it will make data integrity check of the downloaded content. The programs which can do such check-out are many and always free of charge. They are always small (about 0.2-1.0 MB in size) and can be installed as a shell extension (this means one more line(s) in the mouse right-click menu). As for the media files (movie, music, pictures) the data integrity is not significant, for some type of files as archives with installer or disk image files - the data integrity check is crucial. Especially for users with bad or slow connection, or complexed international torrents (ThePirateBay, MiniNova, ISOHunt) - it is recommended always to check out the integrity of the files after completing the torrent.
