4- IMPORTANT! Move/copy the Keygen to the sony setup folder and do #3 a second time.
-=-=-
You should have run the KeyGen and patched files in both ../Sony/Acid Pro 7.0 and ../Sony Setup/Acid Pro 7.0 at this stage.
Vista / Win7 users: If you get an "error has occurred" message, run the KeyGen as Administrator.
-=-=-

6- Choose "Register from another computer" - this will create a small html file on your computer that you can ignore. Don't attempt to register online!
7- When prompted, enter Activation code from #5. You can copy and paste.
-=-=-
Enjoy!
