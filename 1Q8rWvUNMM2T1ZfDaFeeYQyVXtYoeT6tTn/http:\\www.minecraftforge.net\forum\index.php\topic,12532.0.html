Dudes, I have solution for this trouble (for me). Maybe it's helps someone else. FML Installer downloads damaged scala files and JVM cannot open it.

Step 1: Run FML Installer. Check %MC FOLDER%/versions - if installer created "1.6.4-Forge-blah-blah" folder, then move to step 2.

Step 2: Download these files:
http://repo1.maven.org/maven2/org/scala-lang/scala-library/2.10.2/scala-library-2.10.2.jar
http://repo1.maven.org/maven2/org/scala-lang/scala-compiler/2.10.2/scala-compiler-2.10.2.jar

Step 3: Replace files what been downloaded with FML Installer. It placed at %MINECRAFT FOLDER%/libraries/org/scala-lang/....

This solution works with MagicLauncher.
