Firefox "Tilt 3D" Add-on
This addon for Firefox renders web pages in 3d, showing the layers of a web page, how elements are stacked on top of each other. It looks really cool, and you can probably impress the geeky girls. Aside from that, it does have some practical use in figuring out why elements appears as they do. You can find Tilt 3D for Firefox here.
