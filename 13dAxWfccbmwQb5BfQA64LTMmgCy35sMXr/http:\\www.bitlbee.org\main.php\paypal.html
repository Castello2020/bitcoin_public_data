Donate

Since from time to time people ask how they can PayPal some money, I decided to add a link now. This is not going to be a spammy fundraiser, there aren't going to be any blogposts/tweets to announce this!

Maintaining BitlBee mostly costs a lot of time. Most public servers (and web hosting, etc.) are sponsored and don't cost us a cent. There are a few exceptions though, and domain registrations aren't free either. Any money donated may be used for some of these things, maybe to restore the "BitlBee pizza fund", and generally as a little compensation for all time spent on implementing the more boring features. :-)

Donations of any size are welcome. Note that donations are a fine way to bribe us to implement a certain feature, but they're not a guarantee! Donating five euros to get a feature that'd take a whole day (or even more) to implement won't work.
