HDD Low Level Format Tool 4.40

Strona główna Windows Narzędzia Narzędzia dyskowe i plikowe
Pobierz program Asystent pobierania włączony

HDD Low Level Format Tool jest darmowym (do zastosowań domowych) narzędziem do niskopoziomowego formatowania dysków twardych, a także różnego rodzaju kart pamięci i zewnętrznych dysków USB/FireWire.

Galeria programu

  •

Proces niskopoziomowego formatowania nośnika pozwala na trwałe usunięcia przechowywanych na nim danych (nie jest możliwe ich przywrócenie). Ponadto w przypadku starszych dysków twardych operacja ta wykonana musiała zostać, aby przygotować urządzenie do pracy w komputerze. Aktualnie niskopoziomowe formatowanie nośników przeprowadzane jest przez producentów jeszcze w fabryce i pozwala zapisać na nim informacje konieczne do poprawnego działania kontrolera.

Program poza funkcją niskopoziomowego usuwania danych z dysków dostarcza informacje o modelu, wersji firmware itp., a także ogólnej kondycji urządzenia. Poza kasowaniem samych danych HDD Low Level Format Tool usuwa także z dysku wszelakie partycje oraz dane przechowywane w głównym rekordzie startowym MBR.

HDD Low Level Format Tool obsługuje dyski twarde wszystkich wiodących na rynku producentów (m.in. Maxtor, Hitachi, Seagate, Samsung, Toshiba, Fujitsu, IBM, Quantum, Western Digital) połączonych z komputerem za pośrednictwem interfejsów S-ATA (SATA), IDE (E-IDE), SCSI, SAS, USB oraz FIREWIRE. Program daje również możliwość formatowania kart pamięci SD, MMC, MemoryStick i CompactFlash.

Uwaga!

  1. Od wersji 4 producent udostępnił program całkowicie bezpłatnie do użytku domowego. Posiada ono jednak ograniczenie prędkości formatowania, wynoszące 50 MB/s. Chcąc pozbyć się wspomnianego limitu należy wykupić licencję, której cena to 3,30 USD dla użytkowników prywatnych lub 27 USD dla firm.
  2. Do pobrania dostępna jest również wersja portable, nie wymagająca instalacji w systemie i gotowa do użycia tuż po ściągnięciu.
