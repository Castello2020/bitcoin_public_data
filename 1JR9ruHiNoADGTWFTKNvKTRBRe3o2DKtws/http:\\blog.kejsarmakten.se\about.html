About me Blog Projects

I am a software developer, hacker and computer scientist (MS) from Malmö in southern Sweden.

I have a wide technological interest and some favorites include coding, machine learning and security. I often code in either Python or Java and I am now in the process of learning Scala.
Right now I am starting my own company, Sentilyse, working with sentiment analysis solutions. Besides this I am doing some consulting work for FEO media (building machine learning solutions, backend optimizations and security related tasks).

Other interests include fencing (epee), hackathons, meetups and conferances, reading(!), urban gardening, cycling, lock picking and hanging out with my family.

The blog

I used the great Jekyll tool to create this website. Jekyll is an open source static site generator. The blog is also available through the I2P darknet here blog.kejsarmakten.i2p.
