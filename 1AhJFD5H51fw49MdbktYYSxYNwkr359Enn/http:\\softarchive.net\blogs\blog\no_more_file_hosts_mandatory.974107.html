No More File Hosts Mandatory?

SA News


Dear Members and Visitors,

Time to make some changes with file-hosting services.
As you know, our mandatory file hosts are: RapidShare or MediaFire + ANY other mirrors.

Why we want to change it?


Because last week RS and MF has started to suspend many uploaders accounts without clear reason.
And for free users , download speed is very low with RS, limit upload size, throttle upload speed.

As suggested by our members, we will allow uploaders to chose their preferred file hosts.

We choose options :

- All articles MUST provide minimum 3 mirrors + all of file hostings MUST support free download.

- Uploaders MUST not set download links for premium users only.

- Do not delete the old files you have uploaded to RS/MF which was posted on SA before.


When we find good and stable file hosts , sure it's will be in mandatory file-host list. We hope Members and Visitors will co-operate with us for better SoftArchive.

What do you think about it ?

Best regards,
SA Team
