Sothink Tree Menu 3.1.305 | 10.6 MB


Sothink Tree Menu is an application that lets you view and browse every link page on the web site clearly.Programming skills are not necessary, and creating JavaScript tree menu becomes an interesting tour. The browser compatibility is never a puzzle; the tree menu also can be expanded or collapsed from the start.

Sothink Tree Menu supports all major browsers and their different versions on diverse platforms. The supported browsers include Opera7.5+, IE5+, Firefox1.0+, Safari1.3 on Mac and PC. The tree menu can detect the visitors' browser type and load the best script code automatically, which is compatible with browsers on platforms. So you do not worry about the browser compatibility

Built-in templates help you start a professional tree menu from the existing menus. These templates are designed by the experts who engaged in web development. The menu styles are fashionable and versatile; and there must be one or more exquisite templates that suit for your website. To create a new novel tree menu by template, you just do modify the text and replace the image. Furthermore, users are allowed to create their own templates and modify the existing ones.

Sothink Tree Menu provides image library for the users. The image library includes three types of graphics, icon image. button image and background image. If you would like to choose one of them, you can open the image library to select.

Publish Wizard is an intelligent assistance to help you publish the tree menu into web page easily. For beginner, put the cursor in the desired position to insert the tree menu within HTML page automatically; for intermediate, follow the guide to copy and paste the tree menu code by hand; for expert, create JS include file to insert the tree menu, which the tree menu code is included in js file.

Condition makes creating dynamic database-driven tree menus never so easy. Node text, links, tips and other properties can be dynamically generated from database by using JSP, ASP, PHP, ColdFusion or other server side code. Just input variables in the corresponding fields, and write some conditions (if...else..., while, for, etc.) in Conditions panel to generate a tree menu instead of manually coding every line of the tree menu on the page. Furthermore, those conditions have always been saved in configuration file even though modifying the tree menu style.
