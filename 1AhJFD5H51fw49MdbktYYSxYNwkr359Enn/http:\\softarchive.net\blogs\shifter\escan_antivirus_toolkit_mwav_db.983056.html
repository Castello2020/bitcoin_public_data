Antivirus and antispyware scanner. The utility allows you to scan and remove viruses, spyware, adware and other malicious programs that could be infected computer. eScan Antivirus Toolkit does not require installation and can be run directly from anywhere: on your computer, USB media or CD. It can be run even if your computer is already running another antivirus program.

eScan Antivirus Toolkit is updated weekly to the latest version in order to detect newly released spyware and adware, in addition to this mechanism is constantly "improvising", providing a quick and intelligent detection.

Functions eScan Antivirus Toolkit 12.x:
• Complete Scan and clean your computer, as well as providing reports of any viruses detected.
• Check and remove all programs, dialers from your computer.
• Inspection and cleaning of any illegal traffic analyzer, or spyware, adware, keyloggers etc. running in the memory of your computer.
• You can add the program to startup the computer, so that it can scan and clean your computer every time you run.
• When you load the program you will get it together with the latest updated list of viruses that will scan and clean your computer from all known viruses.
• Installation is not required. Just download and run the eScan Antivirus Toolkit to check your computer.
• eScan Antivirus Toolkit runs on Microsoft Windows 2000, XP, Vista, Windows 7 and Windows 2008 (32 and 64 bit), and also supports multiple languages.
