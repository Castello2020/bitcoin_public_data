Since this product is delivered on a usb hard drive, its content is divided into 31 segments. In order to install you need to put the content of each segment into one folder, and then run setup. After you have installed Komplete 9 Ultimate you need to install Unlocked components for Windows or Mac to overwrite original files, otherwise the software will work in demo mode.

You can install not all but only some particular parts of the library. In any case you need to have the first segment which is D1 (see list below) and extract its content into a root folder then you need to extract the content of any other segment into the same root folder, then run setup and select corresponding library components in the installation menu.
