♥☻ Best Archive ☻♥


Debenu Quick PDF Library v9.12 | 36 MB

Quick PDF Library provides a comprehensive set of functions that allow you to create, split, merge, encrypt, modify, manipulate PDF form fields and JavaScript. Images can be added in BMP, TIFF, JPEG, PNG and EMF format. TrueType and Type1 fonts can be embedded, and TrueType fonts can be subsetted allowing Unicode text to be drawn.

Quick PDF Library also includes a full built-in PDF rasterizer allowing PDFs to be rendered to the screen or printer.

Here are some key features of "Quick PDF Library":

· TrueType font support (including font embedding)
· Built-in barcode support
· Import and append to any existing PDF file
· Encryption and decryption (including 128-bit encryption)
· Advanced drawing, including circles and arcs
· Outline and hypertext functions
· Embedded image support
· Rotated text, multi-line paragraph text, word-wrapped paragraph text
· Save-to-stream and Load-from-stream support for web serving
· Page cloning technology
· PDF merging technology
· Form field support
· Advanced page extraction
· Page combining and imposition
