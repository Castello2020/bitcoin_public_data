Click "Activate" - a window, no matter close and run the program again
3. Click "Close" to close the window reaffirm the licensing - the program automatically flips to the top of the activation process (paragraphs 1 and 2). But now, when you click "Activate" appears "correct" window where you must select "I have an activation code Autodesk".
4. Run the keygen (remember about the bit depth of the OS and root!), And in it we press Patch (a must!)

B - click Generate. In the keygen-a (region «Activation»), we obtain the required activation code;
c - the code is inserted in the activation window Autodesk.
