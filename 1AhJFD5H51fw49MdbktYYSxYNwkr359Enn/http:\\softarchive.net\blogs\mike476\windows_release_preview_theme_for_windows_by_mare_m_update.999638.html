This is a most complete and most realistic Windows 8 Release Preview visual style for Windows 7 32-bit and 64-bit on the internet.

This theme is direct port from Windows 8 Release Preview. Over 400 images exported from original Windows 8 aero.msstyles.

I spent a lot of time to all import in this theme.
Almost every part of this theme is original.
I had to change a few things in Photoshop to get more realistic theme.
I found a way to increase caption buttons and reduce the margins of the window.
I also redesigned lone close button to be more in Windows 8 style, start menu is also changed.
With this theme you get real Windows 8 RP color and wallpaper. Color was taken from original theme file.
